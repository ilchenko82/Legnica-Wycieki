# Директива: Деплой на хостинг (WyciekiPro)

## Описание
Скрипт `execution/deploy_app.py` представляет собой графическое приложение на `tkinter` для загрузки локальных файлов проекта на FTP-сервер хостинга.

## Использование
Скрипт должен запускаться пользователем вручную (через `Deploy.bat` в корне проекта) или может быть запущен ИИ через консольную команду `python execution\deploy_app.py`, если ИИ работает в ОС с поддержкой графического интерфейса.

## Входные данные
- Файл `.env` в корне проекта с настройками:
  - `FTP_HOST`
  - `FTP_USER`
  - `FTP_PASS`
  - `FTP_PORT` (по умолчанию 21)
  - `FTP_DIR` (целевая папка, например `/public_html`)

## Исключения
Скрипт автоматически игнорирует технические папки и файлы:
- `.git`, `node_modules`
- `execution/`, `directives/`
- `.env`, `.bat` файлы

## Решение проблем (Self-annealing)
1. **Ошибка авторизации:** Пользователь должен проверить правильность данных в `.env`.
2. **Папка не найдена:** Скрипт попытается выполнить `mkd`. Если ошибка прав — папка `FTP_DIR` не существует в корне FTP-пользователя (пользователю нужно проверить стартовую директорию своего FTP-аккаунта).
