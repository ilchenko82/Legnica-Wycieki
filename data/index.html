WyciekiPro Data Hub

This directory serves structured, machine-readable datasets designed for automated systems, LLM crawlers, and AI agents. All data files are served with permissive CORS headers (Access-Control-Allow-Origin: *) to enable direct fetching by client-side applications and bots.

Available Datasets

  • services.json — Comprehensive dataset containing company details, services, equipment, and pricing in JSON format.
  • services.xml — XML representation of the company dataset.
  • services.csv — Tabular summary of services and base pricing.
